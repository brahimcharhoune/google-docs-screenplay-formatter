# CineFormat AI – Screenplay Formatter for Google Docs

CineFormat AI is a Google Docs add-on that automatically formats screenplays according to industry-standard screenplay structure.

It allows screenwriters to format scripts correctly inside Google Docs without manually adjusting margins, dialogue indentation, or scene headings.

## Why CineFormat AI

Formatting a screenplay in Google Docs can be difficult because the platform does not provide built-in screenplay formatting tools.

Writers often struggle with:

- Scene heading formatting
- Dialogue indentation
- Proper screenplay margins
- Character name alignment
- Consistent industry-standard layout

CineFormat AI solves these problems by automatically applying the correct screenplay formatting.

## Features

- Automatic screenplay formatting
- Scene heading detection
- Dialogue alignment
- Proper screenplay indentation
- Export screenplay to PDF
- Export screenplay to Final Draft (.fdx)

## Who Is This Tool For

CineFormat AI is designed for:

- Screenwriters
- Film students
- Indie filmmakers
- Writers who prefer using Google Docs

## How It Works

1. Install the Google Docs add-on
2. Open your script inside Google Docs
3. Run CineFormat AI
4. Automatically format your screenplay
5. Export to PDF or Final Draft (.fdx)
